# Windows AV2 Builds<br>AOMedia AVM and libavif (with AV2 Support)

Windows `x86_64` binaries of **AVM** and **libavif** built by GitHub Actions.

## Features and Included Components

Every release artifact contains executables and libraries for Windows 64-bit systems:

| Component | Description | Included Binaries / Libraries |
| --- | --- | --- |
| **AVM (AOM Video Model)** | The official research and reference software for the next-generation **AV2** codec. | `avmenc.exe`, `avmdec.exe`, `avm.lib` |
| **libavif** | High-performance library for encoding/decoding **AVIF** images, built with AV2 experimental support enabled. | `avifenc.exe`, `avifdec.exe`, `avif.lib` |

## Obtain the Binaries

Head over to the Releases page and download the zip archives.
